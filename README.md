# Calculadora de Partidas Rankeadas 🎮

Este projeto é a resolução do desafio **"Calculadora de partidas Rankeadas"** proposto na plataforma da DIO.

## 📝 Descrição do Desafio

O objetivo é criar uma função que avalia a quantidade de vitórias e derrotas de um jogador. O programa calcula o "saldo de rankeadas" subtraindo as derrotas das vitórias e determina o nível do jogador com base em suas vitórias, seguindo os critérios abaixo:

- **Ferro**: Vitórias menores ou iguais a 10
- **Bronze**: Vitórias entre 11 e 20
- **Prata**: Vitórias entre 21 e 50
- **Ouro**: Vitórias entre 51 e 80
- **Diamante**: Vitórias entre 81 e 90
- **Lendário**: Vitórias entre 91 e 100
- **Imortal**: Vitórias maiores ou iguais a 101

Ao final, o programa exibe a seguinte mensagem:
`"O Herói tem de saldo de **{saldoVitorias}** está no nível de **{nivel}**"`

## 🛠️ Tecnologias Utilizadas

- **Python**: Linguagem de programação para construir a lógica.
  - Foram utilizados conceitos de:
    - Variáveis
    - Operadores matemáticos e lógicos
    - Laços de repetição (`while`)
    - Estruturas de decisões (`if`, `elif`, `else`)
    - Funções com envio e retorno de parâmetros (`def`)

## 🚀 Como Executar

1. Certifique-se de ter o Python instalado na sua máquina.
2. Abra o terminal ou prompt de comando na pasta do projeto.
3. Digite o seguinte comando:
   ```bash
   python calculadora_rankeadas.py
   ```
4. Siga as instruções no console para inserir as vitórias e derrotas. Caso queira encerrar o programa, basta digitar "sair".
